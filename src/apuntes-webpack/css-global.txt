Primero lo importamos en index.js

import './style.css';

tenemos que descargar una dependencia 

npm install -D mini-css-extract-plugin


configuramos el plugin en el archivo de webpack, podemos usar contentHash para crear nuemeros aleatorios y que no se guarde ne la cache del navegador


npm run build
npm start

y todo craga perfectamente
